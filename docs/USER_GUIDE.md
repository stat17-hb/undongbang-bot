# 🏋️ 운동인증방 사용자 가이드

## 슬래시 커맨드

Discord 채팅창에 `/`를 입력하면 명령어 목록이 표시됩니다.

| 명령어 | 설명 |
|--------|------|
| `/멤버등록` | 운동인증방 멤버로 등록 (최초 1회) |
| `/인증 회차:N` | 운동 인증 (1~3회) |
| `/벌금조회` | 본인 벌금 현황 확인 |
| `/주간현황` | 전체 멤버 주간 현황 |

---

## 사용 방법

### 1️⃣ 멤버 등록
```
/멤버등록
```
- 최초 1회만 하면 됩니다

### 2️⃣ 운동 인증
```
/인증 회차:1
/인증 회차:2
/인증 회차:3
```
- 운동 후 회차를 선택하여 인증
- 인증 사진은 봇 응답에 **답장**으로 첨부

### 3️⃣ 현황 확인
```
/벌금조회    ← 본인만 보임
/주간현황    ← 전체 공개
```
